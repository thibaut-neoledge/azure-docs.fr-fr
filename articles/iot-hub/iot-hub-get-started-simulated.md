# <a name="azure-iot-hub-get-started-with-simulated-devices-tutorials"></a>Didacticiels de prise en main d’appareils simulés avec IoT Hub Azure

Ces tutoriels présentent Azure IoT Hub et les Kits de développement logiciel (SDK) d’appareil. Les tutoriels abordent des scénarios IoT courants pour faire la démonstration des fonctionnalités de IoT Hub. Ils illustrent également la manière dont IoT Hub peut être combiné avec d’autres services et outils Azure pour créer des solutions IoT plus puissantes. Les didacticiels répertoriés dans le tableau ci-dessous montrent comment créer des appareils IoT simulés.

| Langage de programmation |
|----------------------|
| [.NET][Sim_NET]      |
| [Java][Sim_Jav]      |
| [Node.js][Sim_Nd]    |
| [Python][Sim_Pyth]   |

En outre, vous pouvez utiliser une passerelle IoT Edge pour permettre aux appareils simulés de se connecter à votre IoT Hub.

| Langage de programmation | Plateforme           |
|----------------------|------------------- |
| C                    | [Linux][Sim_Lnx]   |
| C                    | [Windows][Sim_Win] |
