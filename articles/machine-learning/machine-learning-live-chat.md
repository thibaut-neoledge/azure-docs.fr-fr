# <a name="get-help-from-machine-learning-live-chat-support"></a>Obtenir de l'aide de la part du support par conversation en direct sur Machine Learning
[Azure Machine Learning Studio](machine-learning-what-is-ml-studio.md) fournit une interface intuitive pour générer des modèles Machine Learning. Il existe une [galerie](machine-learning-gallery-how-to-use-contribute-publish.md) d’exemples fournis par la communauté pour vous aider à commencer. Il existe également un [forum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) où vous pouvez poser des questions. 

Cependant, pour vous débloquer, vous avez parfois besoin de poser une question rapide. Dans [Machine Learning Studio](http://studio.azureml.net/Home), recherchez l’icône de discussion dans le volet de navigation supérieur.  Si cette icône est visible, cela signifie qu'un membre de l'équipe produit est en ligne pour vous apporter une aide en temps réel.

![Discussion Azure ML](./media/machine-learning-live-chat/AzureMLChatNavBar.png)

Tapez votre question pour obtenir une réponse !

![Boîte de dialogue de discussion Azure ML](./media/machine-learning-live-chat/AzureMLChat.png)

**Vous ne voyez pas l’icône Conversation en direct ?**

La conversation en direct est un service assuré par les membres de l’équipe Machine Learning. Si l’icône de conversation en direct ne s’affiche pas, cela signifie que l’équipe n’est pas disponible actuellement. Cela peut être le cas, par exemple, en dehors des heures de travail. 

Vous pouvez toujours publier vos questions sur le [Forum Machine Learning](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning), où elles peuvent être consultées par les membres de l’équipe Machine Learning et d’autres développeurs et scientifiques compétents.
