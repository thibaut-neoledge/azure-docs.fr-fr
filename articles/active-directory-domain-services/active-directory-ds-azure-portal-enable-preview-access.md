# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Activer l’expérience en préversion pour administrer votre domaine managé à partir du portail Azure

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Votre domaine managé actuel n’apparaît pas dans l’expérience du portail Azure (préversion)
Vous disposez d’un domaine managé [Azure AD Domain Services](active-directory-ds-overview.md) qui a été créé à l’aide du portail Azure Classic. Lorsque vous accédez au portail Azure, le panneau Azure AD Domain Services (préversion) ne répertorie pas ce domaine managé.

![Le domaine managé actuel n’est pas affiché dans le portail Azure.](./media/getting-started/missing-domain-services-instance.png)

Cela est dû au fait que votre domaine managé actuel n’a pas été migré vers le nouveau portail Azure. Vous pouvez nous contacter pour activer la nouvelle expérience du portail Azure (préversion) pour votre domaine managé actuel.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Rejoindre la préversion du portail Azure pour Azure AD Domain Services
Suivez les instructions de la page [Contactez-nous](active-directory-ds-contact-us.md) et envoyez-nous un e-mail. Nous sommes heureux d’activer la préversion du portail Azure pour votre domaine managé actuel.

> [!NOTE]
> Lorsque la nouvelle expérience du portail Azure pour Azure AD Domain Services sera généralement disponible, tous les domaines managés actuels seront migrés automatiquement. Vous serez alors en mesure de gérer votre domaine managé existant à partir du portail Azure.
